一、 被 Bing K 站了，或一直没有被收录怎么办？

提交申诉：https://www.bing.com/webmasters/support
申诉方法可以参考这个帖子： 被 Bing K 站了怎么办？（历时整整2个月的完整解决过程）
另外，如果你的网站确实没什么问题的话，第一次申诉失败了，可以过几天再重新提交申诉。
如果你实在搞不定的话，可以来找我帮忙看看。

二、 怎么提高 Bing 的收录速度和效果？

1. 去 Bing Webmaster 提交 sitemap；

2. 使用 IndexNow 提交。

官方文档：https://www.bing.com/indexnow/getstarted
第一步：生成 API Key 并上传到网站根目录下；
第二步：提交 URLs。
我个人常用的方法是通过 API 直接提交向 IndexNow 提交 URLs ：
方法A：自己拼接 curl 或在线生成 curl 命令后在本地电脑终端上运行。 我为了自己使用方便，写了个在线生成 curl 命令的工具（纯前端，服务器不保存数据）：
https://www.lanxk.com/zh/tools/indexnow/
方法B：使用浏览器插件直接提交。 因为上面这个方法要先复制网址到输入框，生成 curl 命令再复制到本地执行又多了一步，所以我就写了个 浏览器插件（Chrome版 / Edge 版）来提高效率，这个插件可以直接在线自动导入 sitemap 里面的网址（可以通过关键词来筛选 URL），直接在线一键提交。
另外也可以自己在网站程序里面写自动提交功能，或通过 CMS 系统的第三方插件来提交。
Cloudflare 也有这个功能，但是我个人不推荐，效果很差。

三、 Bing Webmaster 网站扫描给的配额太低怎么办？能通过申诉提高吗？

首先说答案：无法通过申诉提高。 我有一个网站给的扫描配额只有 10个页面，已经申诉过两次了，每次回复都是说这是系统自动配置的，不能改。
但是有另外一个方法可以试试：删除然后重新添加网站（我暂时还没有试过这个方法）。
