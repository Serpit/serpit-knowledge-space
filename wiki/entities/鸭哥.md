# 鸭哥 (grapeot)

## 简介

AI 领域独立研究者和实践者，superlinear.academy 社区成员。花一年时间构建了 Context Infrastructure 系统，用于突破 LLM 的 [[Consensus天花板]]。

## 主要贡献

- 提出 **Context Infrastructure** 方法论：大量积累 → 分层提炼 → 按需加载
- 开源 [context-infrastructure](https://github.com/grapeot/context-infrastructure)：包含 44 条 axiom、核心 skill、三层记忆系统
- 运营"鸭哥AI手记"（daily.yage.ai）：基于 context 系统的每日 AI 简报
- 发表领域调研报告（superlinear.academy）

## 核心观点

- 决定 AI 产出性质的是 context，不是模型智能
- 个人 context 不贬值，模型智能不断贬值
- 把 bias 显性化本身就有自我认知价值
- [[Mem0]] 等记忆系统蒸馏深度不够（只到事实层），需要深入到判断原则层
